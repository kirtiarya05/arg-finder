ARG Finder is a bioinformatics web application that analyzes DNA sequences and predicts potential antibiotic resistance genes using motif-based detection and biological metrics.

---

##  Features
- FASTA DNA sequence analysis
- Antibiotic resistance motif detection
- GC content calculation
- ORF (Open Reading Frame) prediction
- Resistance scoring model
- Interactive Streamlit dashboard
- Data visualization (bar chart & pie chart)

---

##  Scientific Background
Antibiotic resistance is a major global health concern. Detecting resistance genes in bacterial genomes helps researchers and clinicians understand resistance mechanisms and guide treatment strategies.

This project performs **in-silico screening** of DNA sequences for resistance-associated motifs.
